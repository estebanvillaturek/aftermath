# Aftermath

**Your own words, when they matter.**

A tiny, single-file web app for breaking bad habits — not with willpower, but with memory.

When a habit is about to win, you open Aftermath and read what *you* wrote the last time it happened: how it actually felt afterward. You can walk away, snooze it like an alarm (it comes back, quoting your excuse), or do it anyway — and then capture the aftermath while it's raw, so next time starts with the truth.

All data lives in your browser's local storage. No accounts, no server, no tracking — this repo hosts only the empty app shell.

**Use it:** open the site, then *Add to Home Screen* on your phone. Works offline.
